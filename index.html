🔍 How PrivaCrypt Works & Security Details

🔐 Professional Encryption in Your Browser

PrivaCrypt is a complete cryptographic suite that runs entirely in your browser. No data is ever sent to any server. It uses the native Web Crypto API with standard algorithms like AES-256-GCM, PBKDF2 key derivation, and RSA-2048 key generation.

✨ Core Features

📖 Book Cipher: Coordinate-based encryption using a base text. Each character is encoded as a word position or X:Y coordinate.

⚡ AES-256-GCM: Military-grade encryption with random salt, unique IV, and 250,000 PBKDF2 iterations.

🌐 LSB Steganography: Hides messages in PNG images by modifying the least significant bit. Includes SHA-256 checksum.

📜 Merkle Blockchain: Seals files with a Merkle root to verify integrity.

🛡️ Why PrivaCrypt?

Unlike other online tools, PrivaCrypt never uploads your data. All processing happens locally on your device using pure JavaScript and the Web Crypto API. 100% open source. Compatible with all modern browsers.
