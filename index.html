PrivaCrypt

Cifrado militar AES-256 · Sistema Libro · Esteganografía · 100% Cliente · Open Source

🔐 Herramienta de Cifrado Profesional en tu Navegador

PrivaCrypt es una suite criptográfica completa que funciona completamente en el navegador, sin necesidad de registro ni envío de datos a servidores externos. Utiliza la Web Crypto API nativa para garantizar la máxima seguridad con algoritmos estándar como AES-256-GCM, derivación de claves PBKDF2 y generación de pares RSA-2048.

✨ Funcionalidades Principales

📖 Sistema LibroCifrado por coordenadas basado en un texto base. Cada carácter se codifica como posición de palabra o coordenada X:Y.
⚡ AES-256-GCMEl estándar de cifrado militar. Con salt aleatorio, IV único y 250,000 iteraciones PBKDF2.
🌐 Esteganografía LSBOculta mensajes en imágenes PNG modificando el bit menos significativo. Incluye checksum SHA-256.
📜 Blockchain MerkleSella archivos con raíz Merkle para verificar su integridad.

🛡️ ¿Por qué elegir PrivaCrypt?

A diferencia de otras herramientas online, PrivaCrypt nunca envía tus datos a ningún servidor. Todo el procesamiento ocurre localmente en tu dispositivo usando JavaScript puro y la Web Crypto API.

Totalmente open source. Disponible en GitHub Pages. Compatible con todos los navegadores modernos.
